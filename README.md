# rock-paper-scissor

>This project consists of a simple game called *Rock,Paper & Scissor*

# rules
 > This is a FIVE round game.
 >>  **Rock** beats **Scissor**.
 >>> **Scissor** beats **Paper**.
 >>>>**Paper** beats **Rock**.
 # Game Support
 > The game now supports all kind of devices.
